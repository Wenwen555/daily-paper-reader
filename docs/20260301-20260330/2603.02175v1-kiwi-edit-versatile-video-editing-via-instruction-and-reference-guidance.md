## 摘要
基于指令的视频编辑技术发展迅速，但现有方法常因自然语言在描述复杂视觉细节上的固有局限而难以实现精确的视觉控制。尽管参考引导编辑提供了稳健的解决方案，但其潜力目前受限于高质量配对训练数据的稀缺。为弥合这一差距，我们引入了一个可扩展的数据生成流程，该流程利用图像生成模型创建合成的参考支架，将现有的视频编辑对转换为高保真的训练四元组。基于此流程，我们构建了RefVIE——一个专为指令-参考跟随任务定制的大规模数据集，并建立了用于全面评估的RefVIE-Bench。此外，我们提出了一种统一的编辑架构Kiwi-Edit，它通过可学习查询与潜在视觉特征的协同作用来实现参考语义引导。我们的模型通过渐进式多阶段训练课程，在指令跟随和参考保真度方面取得了显著提升。大量实验表明，我们的数据与架构在可控视频编辑领域确立了新的最先进水平。所有数据集、模型及代码均已发布于https://github.com/showlab/Kiwi-Edit。

## Abstract
Instruction-based video editing has witnessed rapid progress, yet current methods often struggle with precise visual control, as natural language is inherently limited in describing complex visual nuances. Although reference-guided editing offers a robust solution, its potential is currently bottlenecked by the scarcity of high-quality paired training data. To bridge this gap, we introduce a scalable data generation pipeline that transforms existing video editing pairs into high-fidelity training quadruplets, leveraging image generative models to create synthesized reference scaffolds. Using this pipeline, we construct RefVIE, a large-scale dataset tailored for instruction-reference-following tasks, and establish RefVIE-Bench for comprehensive evaluation. Furthermore, we propose a unified editing architecture, Kiwi-Edit, that synergizes learnable queries and latent visual features for reference semantic guidance. Our model achieves significant gains in instruction following and reference fidelity via a progressive multi-stage training curriculum. Extensive experiments demonstrate that our data and architecture establish a new state-of-the-art in controllable video editing. All datasets, models, and code is released at https://github.com/showlab/Kiwi-Edit.

## 2. 方法论
### 2.1 核心思想
1.  **数据生成流水线 (RefVIE)**：既然缺乏现成的四元组数据，就利用强大的预训练图像生成模型作为“参考合成器”，将已有的（源视频，指令，目标视频）三元组自动扩展为四元组。关键在于从目标视频中提取出“被编辑区域”作为参考图像。
2.  **统一架构 (Kiwi-Edit)**：设计一个能够融合多模态条件（源视频、文本指令、可选参考图像）的统一框架。其核心是结合语义理解（通过MLLM）和结构/纹理控制（通过DiT中的混合注入策略）。

### 2.2 关键技术细节
*   **RefVIE数据集构建流程**：
    *   **阶段1 - 源聚合与过滤**：汇集多个开源指令编辑数据集（Ditto, ReCo, OpenVE），使用`EditScore`进行质量过滤。
    *   **阶段2 - 定位与分割**：使用`Qwen3-VL`根据指令在目标视频首帧中定位编辑区域，并用`SAM3`生成精确掩码。
    *   **阶段3 - 参考图像合成**：使用图像编辑模型`Qwen-Image-Edit`，根据掩码从目标帧中提取被编辑对象或背景，生成干净的参考图像。
    *   **阶段4 - 质量控制与去重**：使用MLLM验证语义一致性，并使用CLIP特征进行全局去重。最终从370万初始样本中得到47.7万高质量四元组。

*   **Kiwi-Edit模型架构与训练策略**：
    *   **架构组成**：
        *   **冻结的MLLM (`Qwen2.5-VL-3B`)**: 编码多模态输入序列。通过LoRA微调适应视频编辑任务。
        *   **双连接器机制**: 
            *   `Query Connector`: 将一组可学习的查询令牌投影到DiT维度，用于提炼抽象的“编辑意图”。
            *   `Latent Connector`: 提取参考图像的视觉特征令牌。
        *   **扩散Transformer (DiT) (`Wan-TI2V-5B`)**: 主干生成模型。
    *   **混合潜在注入策略 (关键创新)**:
        *   **源视频控制 (逐元素加法)**: VAE编码的源视频潜在特征经过一个零初始化的PatchEmbed层后，与噪声潜在相加。此操作由一个可学习的、时间步相关的标量 `γ(t)`调制，以在保留原始时空结构的同时允许灵活编辑。
        *   **参考图像控制 (序列拼接)**: VAE编码的参考图像潜在特征被拼接到DiT的输入序列中。这使得模型可以通过空间注意力机制直接“复制”参考图像的纹理细节。
    *   **渐进式三阶段训练课程**:
        1.  `MLLM-DiT对齐`:冻结主干网络和DiT权重，仅训练连接器和查询令牌等桥接组件。使用纯文本的图像/视频三元组数据建立语义映射。
        2.  `指令微调`:解冻DiT层进行联合优化。在纯文本的视频三元组上进行训练以学习通用编辑能力。采用分辨率课程学习（从低分辨率开始）。
        3.  `参考引导精调`:引入新构建的RefVIE四元组数据进行混合训练（同时包含纯指令和带参考的数据），使模型学会利用视觉参考进行精细纹理迁移。
